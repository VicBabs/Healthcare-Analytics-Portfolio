# Healthcare Power BI Dashboard

## Project Overview
This project analyzes healthcare data using Microsoft Power BI to identify trends in patient demographics, healthcare utilization, and key performance indicators.

- Excel
- Data Visualization

## Key Insights
- Identified trends across patient groups.
- Visualized healthcare performance metrics.
- Created interactive dashboards for decision-making.
